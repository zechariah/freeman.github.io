JavaScript Notes (2) @npm config

npm

  • npm config

  • Install a package that is sitting in a folder on the filesystem:
    npm install <folder>

  • npm help install, npm help list

  • List installed packages:
    npm list -g

  • cnpm: npm client for cnpmjs.org

  • $ npm install cnpm -g --registry=https://registry.npm.taobao.org

npmbox

  • Create an archive for offline installation of the given package:
    npmbox -v <package> <package>...

  • Extracts a .npmbox file and installs the contained package:
    npmunbox -vg <nmpbox-file> <npmbox-file>...

  • Install npmbox from an .npmbox file:
    npm install --global --cache ./.npmbox-cache --optional --no-registry --fetch-retries 0 --fetch-retry-factor 0 --fetch-retry-mintimeout 1 --fetch-retry-maxtimeout 2 npmbox

GPG Notes

GPG

  • Key server list:
    — keyserver.ubuntu.com
    — pgp.mit.edu
    — http://pgp.mit.edu:11371
    — hkp://keys.gnupg.net
    — OpenPGP Keyserver: http://subkeys.pgp.net:11371/
    — subkeys.pgp.net
    — www.gpg-keyserver.de

  • Search public key from a key server:
    gpg --keyserver subkeys.pgp.net --search-keys [key ID]

  • Import public key from a key server:
    gpg2 --keyserver hkp://pgp.mit.edu:11371 --recv-keys 0xB88CDF85

  • Import/Merge keys: gpg --import [key files]

  • Show fingerprints: gpg2 --fingerprint 0xB88CDF85

  • Verify a signature: gpg --verify [signature file(.sig or .asc)]

  • List keys: gpg -k

  • Export public key: gpg -a --export [key ID]

EasyPG Assistant

EasyPG Assistant is an Emacs user interface to GNU Privacy Guard (GnuPG).

EasyPG Assistant is a part of the package called EasyPG, an all-in-one GnuPG interface for Emacs.

  • Verify file: epa-verify-file

Note-taking Solutions

Emacs Notes (2) @howm

Wiki-like notetaking tool

howm: Write fragmentarily and read collectively.

== Write a note

  • Type C-c , c to create new note.

== Concatenate notes

  • Type C-c , , to open the menu

  • Move the cursor to [All] and type [return]

  • Type @ to concatenate all notes

  • Type [space] and [backspace] to scroll the notes.

  • Type @ to toggle summary-view and concatenated-view.

!! apvlv
apvlv is a PDF/DJVU/UMD/TXT Viewer Under Linux/WIN32 and its behaviour like Vim.

Like Vim, Apvlv makes people reading their PDF files just like using Vim.

  • [num]C-f — to forward page
  • [num]C-b — to previous page
  • 0 — show the content of a directory on your system
  • t— open the slected document in a new tab

!! Vimperator

Vimperator is a Firefox browser extension with strong inspiration from the Vim text editor,
with a mind towards faster and more efficient browsing.

  • d —- Delete current buffer tab
  • :set gui=none
  • :set gui=menu
  • <space>/<C-b> —- scroll down/up by one page
  • <C-d>/<C-u> —- scroll down/up by 1/2 page
  • [count]gT —- Go to the previous tab
  • [count]gt —- Go to the next tab
  • c —- Start caret mode
