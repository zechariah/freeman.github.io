Ruby Notes (1) @How to Install Ruby Gems

RubyGems

RubyGems is a package management framework for Ruby.

  • Add gem sources:
  • Install the gems on the destination machine from the local files:
    gem install --force --local *.gem
    Or
    gem install --local *.gem

Bundler

Bundler provides a consistent environment for Ruby projects by tracking and installing the exact gems and versions that are needed.

  • Generate a Gemfile with the default rubygems.org source by running bundle init.

  • Gemfile is a format for describing gem dependencies for Ruby Programs.

    Place the Gemfile in the root of the directory containing the
    associated code.

  • bundle install

  • bundle install --without=pygments jekyll

  • bundle install --local

  • bundle exec ruhoh server 9292

  • Write Gemfile from the command-line:

My Gemfile

  • For MarkdownJournal
  • For Ruhoh

Install DevKit

  • Run Installation Scripts
    First ruby dk.rb init then ruby dk.rb install
  • Test Installation
    Confirm your Ruby environment is correctly using the DevKit by running gem install json --platform=ruby. JSON should install correctly and you should see with native extensions in the screen messages.
    Next run ruby -rubygems -e "require 'json'; puts JSON.load('[42]').inspect" to confirm that the json gem is working.

Pandoc Notes

Pandoc Notes

  • HTML with CSS:
    pandoc 01-chapter2.markdown -o chapter2.html -c Github.css
    — http://www.yangzhiping.com/tech/pandoc.html

  • HTML with smart quotes, table of contents, CSS, and custom footer:
    pandoc -s -S --toc -c pandoc.css -A footer.html README -o example3.html

  • HTML with embedded CSS:
    pandoc -s -S --toc -H pandoc.css -A footer.html README -o example3.html

  • pandoc -s -o output.html input1.md intput2.md intput3.md

  • Syntax highlighting of delimited code blocks:
    pandoc code.text -s --highlight-style zenburn -o example18g.html

  • Markdown citations:
    pandoc -s -S --biblio biblio.bib --csl mhra.csl CITATIONS -o example24b.html

  • Textile reader:
    pandoc -s -S example25.textile -f textile -t html -o example26.html

  • From HTML to Markdown:
    pandoc -s -r html example.html -o example.md

  • If your local character encoding is not UTF-8, you should pipe input and output through iconv:
    iconv -t utf-8 input.txt | pandoc | iconv -f utf-8

Reference:

  1. http://johnmacfarlane.net/pandoc/demos.html
  2. Tip: using Pandoc to create truly standalone HTML files
  3. Markdown写作进阶：Pandoc入门浅谈

JavaScript Notes (1) @How to Install Node Programs

Install node programs

  • npm config(?what is npm?)

  • Install cnpm(?what is cnpm?)
    npm i -g cnpm

  • Examples
    npm i -g harp --registry=http://r.cnpmjs.org

    npm i -g grunt-cli grunt-init --registry=http://r.cnpmjs.org

Install node programs with npmbox

npm addon utility for creating and installing from an archive file of an npm install, including dependencies. This lets you create a “box” of an installable package and move it to an offline system that will only install from that box.

  • Usage of npmbox

You must specify at least one package.

You can specify more than one package and a separate archive will be created for each specified.

npmbox files end with the .npmbox extension.

  • Usage of npmunbox

You must specify at least one file.

You may specify more than one file, and each will be installed.

  • Installing npmbox from an .npmbox file

    npm install --global --cache ./.npmbox-cache --optional --no-registry --fetch-retries 0 --fetch-retry-factor 0 --fetch-retry-mintimeout 1 --fetch-retry-maxtimeout 2 npmbox

Node Wiki

A simple git based wiki system for markdown files written in node.js.

  • Install
    cnpm i nodewiki -g

  • Usage
    nodewiki [options]

    If you do not want other computers to be able to access node wiki, then use --local or --port=127.0.0.1. The URL for node wiki will then be http://localhost:8888/.
