Emacs Notes (1)

  • C-x C-d —- List directory (brief)

  • C-x C-t —- ?

  • M-x org-time-stamp —- insert date

  • C-u M-x org-time-stamp —- insert date time

  • difference between load-file and load?

  • Insert time with insert-time.el —- C-c d and C-c t

  • Load a file
    (load-file “insert-time.el”)

(load “c:/bjk/elisp/bjk-timestamp.el”)

  • Adding a directory

  • M-X join-line:
    Join this line to previous and fix up whitespace at join

  • M-X info-emacs-manual —- View emacs manual;
    C-h r —- Read the Emacs Manual;

  • C-h i d —- The Info Directory

  • Query replace: M-%
    replace this one, go on to next with SPC
    replace all remaining matches with !

  • Select coding system (?)

  • C-x o — switch cursor to anthor window

  • C-l — Clear screen and redisplay everything putting the text near the cursor at the center.

  • Motion 

    entity to move over              backward        forward
character                        C-b             C-f
word                             M-b             M-f
sentence                         M-a             M-e
go to buffer begining (or end)   M-<             M->

  • C-u 0 C-l — redraw the screen, putting the current line at the top

  • Killing and Deleting

    entity to kill                        backward    forward
character(delete, not kill)                  DEL        C-d
word                                M-DEL       M-d
line(to end of)                        M-0 C-k     C-k
yank back last thing killed                   C-y
replace last yank with previous kill         M-y

  • C-x u — Undo

  • Keyboard Macros

    • C-x ( — start defining a keyboard macro
    • C-x ) — end keyboard macro definition
    • C-x e — excute last-defined keyboard macro

  • M-x ediff —- Compare two files and highlight their differences

  • M-x gnus —- Read news, mail, and RSS feeds

  • C-h t —- Emacs Tutorial;

    C-h m —- get mode-specific information;
    C-h k —- Shows documentation for the command associated with any particular key;
    C-h f —- Shows documentation for any particular command, by name (i.e. what you would type after M-x );
    C-h a —- Search for commands by keywords or regexp
    C-h C-f —- GNU Emacs FAQ

  • M-x auto-fill-mode —- Wraps your lines automatically when they get longer than 70 characters.

  • M-x list-packages

  • M-x linum-mode —- Display line number

  • M-x webjump
